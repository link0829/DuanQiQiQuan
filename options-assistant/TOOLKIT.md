## 二、工具详解（按使用顺序）

### ① 大盘四指数 — [方向定调]
**命令：** 自动执行
**作用：** 看今天市场整体在干嘛
```
SPY 标普（全市场）→ 涨/跌
QQQ 纳斯达克（科技）→ 涨/跌
DIA 道琼斯（蓝筹）→ 涨/跌
IWM 罗素（小盘）→ 涨/跌
```
**用法：** 哪几个同步就是今天的方向

### ② 宏观偏见 — [日线定多空]
**命令：** 自动执行
**作用：** MA20/60/120排列 + RSI
```
多头排列(20>60>120) → 偏多
空头排列(20<60<120) → 偏空
粘合 → 观望
```

### ③ 市场结构 — [当前趋势]
**命令：** 自动执行
**作用：** HH/HL/LH/LL + BuOS/BdOS
```
多头趋势 → 只做多
空头趋势 → 只做空
震荡/RANGE → 观望
```

### ④ 斐波那契 — [画空间]
**命令：** 自动执行（24根5分K）
**作用：** 找支撑/阻力/OTE（Optimal Trade Entry）
```
0.0   = 波段顶
0.382 = 浅回调
0.5   = 中位
0.618 = ★ OTE（最佳进场位）
0.786 = 深回调
1.0   = 波段底
```
**用法：** 价格到0.618附近+其他工具共振 = 进场信号

### ⑤ OB结构带 — [供需区]
**命令：** 自动执行
**作用：** 找大资金建仓区
```
黄色支撑带 = 暴涨前最后那根阴线实体区域（需求区）
头顶压力区 = 暴跌前最后那根阳线实体区域（供给区）
```
**用法：** 结合Fib 0.618用，两个重合 = 强共振

### ⑥ FVG几何+CE — [动能加速]
**命令：** 自动执行
**作用：** 检测是否有Fair Value Gap缺口
```
Bearish FVG = K1_Low > K3_High（下跌加速）
Bullish FVG = K3_Low > K1_High（上涨加速）
CE = FVG的中点（0.5位）→ 价格测试CE收针=极优买点
```
**用法：** 有FVG=趋势在加速，0DTE黄金窗口

### ⑦ ICT流动性 — [防假破]
**命令：** 自动执行
**作用：** 检测BSL/SSL扫荡
```
BSL飙高后跌回 = 上方流动性被扫 → 警惕假突破
SSL砸低后弹回 = 下方流动性被扫 → 警惕假跌破
```
**用法：** 配合反手协议用，假突破后立刻反方向做

### ⑧ Pin Bar评分 — [形态确认]
**命令：** 自动执行
**作用：** Pin Bar强度评分1-5
```
4-5分 = 强信号
2-3分 = 中等
1分 = 弱
```

### ⑨ 0DTE风控 — [管住手]
**命令：** `python3 risk_manager.py check XX CALL 3 1.50`
**作用：** 检查是否爆仓
```
余额检查 / 单笔30% / 日亏限额 / 连亏3次
```

## 三、交易计划格式

```
📋 交易计划 — TSLA PUT

  触发条件:
    ① 价格在Fib 0.618 ($XX)
    ② OB支撑带 ($XX~$XX)
    ③ 出现SSL扫荡+Pin Bar 4/5

  进场: 满足①②③后，下一根1分K开盘市价

  止损:
    价格止损: 跌破信号K线低点 $XX
    时间止损: 进场8分钟未脱离0.618区

  止盈:
    第一抛售(50%): 反弹到0.382 或 FVG压力
    第二抛售(50%): 推损追踪 → BSL触发区+CVD走平
```
