## 📐 斐波那契补充规则

| # | 场景 | 操作 |
|---|------|------|
| ① | 急跌反弹到0.618 + 启明星K线 | **做多**，目标 **0.382** |
| ② | 五浪末端价格到 1.618 + MACD顶背离 | **反手做空** |
| ② | 五浪末端价格到 1.618 + MACD底背离 | **反手做多** |
| ③ | 横盘 0.382 压力位+放量突破 | **跟进做多** |
| ④ | 0.5 易假突破，站稳 0.786 才是强势 | **确认后加仓** |
| ⑤ | 0.618 做多，止损放 **0.786 下方** | **严格止损** |

## 📈 多EMA系统

| EMA | 作用 | 规则 |
|-----|------|------|
| **8 EMA** | 强弱判断 | 线上强势，线下弱势 |
| **21 EMA** | 买卖点 | 线上找买点，线下找卖点 |
| **55 EMA** | **定多空** | 线上多头，线下空头 |
| **100 EMA** | HSS基准 | HSS多空方向判定 |
| **144 EMA** | 大趋势 | 决定整体方向 |

### EMA 操盘规则
- **跌破 21 EMA → 减仓**
- **跌破 55 EMA → 清仓**
- 55 EMA 线上只做多，线下只做空
- 8/21 金叉死叉辅助判断
