### 趋势日吃法 — 盈利移动止损策略 (来自6.29 TSLA复盘)

核心逻辑：
- 开盘V转确认趋势日 → 第一时间上车
- 成本设为保底线（亏了就砍）
- 每涨一波就移动止损
- 只要趋势不破就让它跑
- 趋势结束/放量滞涨 → 止盈

优势：
- 最大亏损 = 入场时的风险（极小）
- 潜在收益 = 全天趋势（巨大）
- 心理负担小（永远不会亏钱持仓）

适用条件：
- 必须是真趋势日（不是震荡日）
- 开盘30分钟内判断出来
- 持续放量推升确认

## 2026-07-08 — 新增交易规则：3分钟VWAP突破

### VWAP突破进场（做多版）
1. 3分钟K线第一根实体冲破VWAP
2. 第二根K线不弱于第一根（继续同向推进）
3. RSI确认（不背离，方向一致）
→ 触发：市价做多CALL

### VWAP跌破进场（做空版）
1. 3分钟K线第一根实体跌破VWAP
2. 第二根K线继续向下，不弱于第一根
3. RSI确认
→ 触发：市价做空PUT

### 意义
- 3分钟比1分钟稳，比5分钟快
- VWAP = 日内多空分界线
- 连续两根确认 = 不是假突破
- RSI加持 = 动能配合

## 2026-07-10 — 重要教训：消息面驱动 > 纯技术面

### META 7/9 暴涨的真正原因

**背景：**
META 7/8收$603 → 7/9开$584→探$577→收$631（+$54, +9%）

**消息面真相：**
```
第一阶段 — 机构恶意解读：
  META宣布出售算力 → 机构解读为"AI需求不足，在甩卖资产"
  → 散户恐慌卖出 → META砸到$577

第二阶段 — 扎克伯格澄清：
  "出售的是过剩算力，不是核心资产"
  "过剩算力出售还能增加一大笔收入"（利好！）
  → 空头踩踏 → META冲到$633
```

**教训：**
```
机构利用信息不对称制造波动：
  ① 释放模糊信息 → 恶意解读 → 砸盘
  ② 等散户割肉 → 低位接盘
  ③ 澄清真相 → 拉高出货

我们如果只看技术面:
  看到$577→SSL扫荡→VWAP突破→可以进
  但不知道为什么跌，就不敢进

以后分析必须加一条:
  📰 消息面扫描：最近有什么新闻/解读/澄清？
```

### 规则更新
- 分析个股前先搜一下最近3天的相关新闻
- 重大消息出台后24小时内的暴跌 → 可能是恶意解读
- 等公司官方回应 → 往往反转
